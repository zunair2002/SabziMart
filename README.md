# Grocery Delivery App

A **modern, full-featured grocery delivery platform** built with **Next.js** and **React**, designed for scalability, performance, and real-world use cases.  
The application provides **secure authentication, real-time delivery tracking, role-based access control**, and optimized data handling using caching and cloud services.  
It focuses on delivering a **smooth user experience for customers, vendors, and admins** in a single unified system.

---

## **Features**

- **Next.js & React**: Fast and responsive frontend with server-side rendering.
- **Authentication**: Secure login & registration using **NextAuth.js**.
- **Password Security**: Passwords hashed before storing.
- **Role-Based Access Control (RDAC)**: Admin, Vendor, and Customer roles with different permissions.
- **Live Map Tracking**: Track deliveries in real-time.
- **Image Uploads**: Product images uploaded to **Cloudinary**.
- **Redis**: Used for caching session and live data.
- **Database**: MongoDB / MySQL (can be adapted based on your preference).

---

## **Tech Stack**

- **Frontend**: React + Next.js  
- **Authentication**: NextAuth.js  
- **Database**: MongoDB / MySQL  
- **Cache**: Redis  
- **Image Storage**: Cloudinary  
- **Other**: Node.js, Express API routes, Tailwind CSS
